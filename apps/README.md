# Apps do native-demo-app

Baixe os arquivos em https://github.com/webdriverio/native-demo-app/releases e coloque nesta pasta:

- **Android**: `android.wdio.native.app.v2.0.0.apk`
- **iOS**: `ios.simulator.wdio.native.app.v2.0.0.zip`

Ajuste o nome do arquivo nas configs `config/wdio.android.app.conf.js` e `config/wdio.ios.app.conf.js` se usar outra versão.
